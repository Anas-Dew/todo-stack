Todo Stack Quick Install

Manage tasks without ever leaving your browser's tab

With a minimalistic interface and a focus on keyboard shortcuts, you can quickly add, complete, and undo tasks without ever leaving your browser’s tab.

Start with Control 

                        1. Add a Task
                        CTRL + Enter

                        2. Complete a Task
                        CTRL + \

                        3. Undo a task
                        CTRL + '
                        
                        4. Exit view
                        ESC

                        Optional. Complete a Selected Task
                        CTRL + ]
